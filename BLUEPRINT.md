# Project: Berlin Naija Foodie Web Presence

## Goal
Transition from "DM for orders" to a streamlined digital storefront that handles takeout, nationwide shipping, and catering inquiries.

## 1. Strategy (COO Perspective)
- **Friction Point:** DM-based ordering is a bottleneck.
- **Conversion Path:** Instagram → Website → Order/Payment → Fulfillment.
- **Revenue Streams:**
    - Local Takeout (Berlin).
    - Nationwide Shipping (Ships across 🇩🇪).
    - Bespoke Corporate/Event Catering.

## 2. Site Map & Architecture
- **Home:** High-energy visuals, clear CTAs ("Order Now", "Book Catering").
- **Menu (Dynamic):** Categories for Halal, Vegan/Vegetarian, and Bulk options.
- **Ordering System:**
    - Local: Pickup scheduler.
    - Shipping: Integration with German carriers (DHL/Hermes).
- **Catering Request Form:** Structured lead-gen form for corporate/event inquiries.
- **About/Our Story:** Highlighting "Locally sourced ingredients from 9ja".

## 3. Tech Stack (Architect Perspective)
- **Frontend:** Next.js + Tailwind CSS.
- **Backend/CMS:** Shopify (E-commerce focus) or Sanity.io (Custom curated experience).
- **Payments:** Stripe.
- **Deployment:** Vercel.
