# Project: Berlin Naija Foodie - Prototype Files

## 1. Folder Structure
/prototype
  /frontend (Next.js)
  /backend (Strapi config/schema)
  /assets (Placeholders for food images)

## 2. Implementation Details
- **Frontend:** Will be a clean, mobile-first Next.js app.
- **Backend:** Strapi with a `FoodItem` collection (name, description, price, image, category).
- **Payment:** Stripe integration for the checkout flow.
